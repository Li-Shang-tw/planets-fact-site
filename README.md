# planets-fact

## 專案說明:
用Vue、vite與Vue Router製作的SPA,可以透過導覽列與按鈕切換瀏覽八大行星的總覽(overview)、內部結構(internal structure)、外部地理(surface geology)等不同資訊

## Build With 
-  Vue
- Vue Router

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
